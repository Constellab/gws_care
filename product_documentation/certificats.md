# Certificats médicaux

## Rôles autorisés
- Médecin

## À quoi ça sert
Les certificats médicaux formalisent les conclusions cliniques : aptitude au poste, arrêt de travail, visite de pré-embauche, Suivi Individuel Renforcé (SIR), vaccination, etc.

## Types de certificats disponibles
- Certificat d'aptitude
- Arrêt de travail
- Visite d'embauche
- Visite périodique
- Accident du travail / Maladie professionnelle
- Suivi Individuel Renforcé (SIR)
- Vaccination

## Comment l'utiliser
1. Depuis le [détail d'une consultation](detail-consultation.md) ou la fiche patient, cliquez sur **Nouveau certificat**.
2. Sélectionnez le type de certificat dans la liste.
3. Renseignez la date d'émission.
4. Renseignez la conclusion médicale (par exemple : apte / inapte au travail).
5. Ajoutez les restrictions ou réserves éventuelles.
6. Selon le type sélectionné, complétez les champs spécifiques (dates de début/fin, date de reprise, zone corporelle affectée, type d'exposition, nom et lot du vaccin, etc.).
7. Cliquez sur **Enregistrer le certificat**. Le PDF est généré immédiatement.

## Astuces et points d'attention
- Le certificat peut être envoyé par email au patient via le bouton **Envoyer le PDF par email au patient**.
- Les certificats peuvent être archivés de la même façon que les ordonnances (voir [ordonnances.md](ordonnances.md)).
