# Consultations

## Rôles autorisés
- Opérateur
- Médecin

## À quoi ça sert
La page Consultations liste les visites de type consultation médicale réalisées, avec leurs résultats et leurs validations.

## Comment l'utiliser
1. Cliquez sur **Consultations** dans le menu de gauche.
2. Filtrez par date, médecin ou statut de validation pour retrouver une consultation.
3. Cliquez sur une ligne pour ouvrir le détail de la consultation : examens réalisés, ordonnances, certificats émis et interprétations médicales.

## Astuces et points d'attention
- Les opérateurs peuvent consulter les consultations ; seuls les médecins peuvent saisir des résultats et valider une interprétation médicale.
