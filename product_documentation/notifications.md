# Notifications

## Rôles autorisés
- Opérateur
- Médecin
- Admin

## À quoi ça sert
Le module Notifications permet d'envoyer des messages aux patients (individuellement ou via leur compte) et aux médecins. Les messages entrants s'affichent dans la boîte de réception.

## Comment l'utiliser

### Envoyer un message
1. Cliquez sur **Notifications** dans le menu de gauche, puis sur l'onglet **Composer**.
2. Sélectionnez le destinataire : Patient, Compte, ou Médecin.
3. Sélectionnez le canal d'envoi : Email, SMS, ou WhatsApp.
4. Renseignez l'objet et le contenu du message.
5. Cliquez sur **Envoyer le message**.

### Consulter et répondre aux messages reçus
1. Ouvrez l'onglet **Reçus**.
2. Cliquez sur un message pour le lire.
3. Cliquez sur **Répondre** pour envoyer une réponse directement liée à ce message.

## Astuces et points d'attention
- L'icône cloche dans la barre latérale affiche en permanence le nombre de messages non lus.
- Envoyer un message à un compte entreprise notifie tous les patients liés à ce compte ayant une adresse email renseignée.
