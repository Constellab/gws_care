# Gestion des documents

## Rôles autorisés
- Opérateur
- Médecin
- Admin

## À quoi ça sert
La page Documents regroupe tous les fichiers téléversés pour l'ensemble des patients : rapports médicaux, analyses, ordonnances scannées, radiographies, etc.

## Comment l'utiliser
1. Cliquez sur **Documents** dans le menu de gauche.
2. Filtrez par patient, type de document (ordonnance, certificat, rapport…) ou période.
3. Cliquez sur un document pour ouvrir l'aperçu intégré.
4. Cliquez sur **Télécharger le fichier** pour récupérer le fichier original.

## Astuces et points d'attention
- Pour ajouter un nouveau document, voir [televerser-document.md](televerser-document.md).
