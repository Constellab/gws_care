# Mes ordonnances

## Rôles autorisés
- Patient

## À quoi ça sert
La page Mes ordonnances liste toutes les ordonnances émises lors de vos consultations, avec la date, le médecin prescripteur et la liste des médicaments.

## Comment l'utiliser

### Télécharger une ordonnance
1. Cliquez sur **Mes ordonnances** dans le menu de gauche.
2. Cliquez sur l'ordonnance souhaitée.
3. Cliquez sur **Télécharger le fichier** pour obtenir le PDF.

### Recevoir l'ordonnance par email
1. Depuis la même page, utilisez le bouton **Envoyer le PDF par email** si vous souhaitez la recevoir directement dans votre boîte mail (peut nécessiter de contacter l'équipe médicale via [mes-notifications.md](mes-notifications.md)).
