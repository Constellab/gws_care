# Mes consultations

## Rôles autorisés
- Patient

## À quoi ça sert
La page Mes consultations liste toutes vos consultations médicales réalisées, avec la date, le médecin responsable et le statut.

## Comment l'utiliser
1. Cliquez sur **Mes consultations** dans le menu de gauche.
2. Choisissez l'affichage souhaité : vue **Liste** (chronologique) ou vue **Calendrier**.
3. Cliquez sur une consultation pour en voir le détail.
