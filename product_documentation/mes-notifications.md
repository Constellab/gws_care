# Mes notifications

## Rôles autorisés
- Patient

## À quoi ça sert
La messagerie patient vous permet d'échanger des messages avec l'équipe médicale de manière sécurisée : rappels de rendez-vous, résultats, communications de l'équipe.

## Comment l'utiliser

### Répondre à un message reçu
1. Cliquez sur **Mes notifications** dans le menu de gauche, onglet **Reçus**.
2. Cliquez sur le message à consulter.
3. Cliquez sur **Répondre** pour envoyer une réponse directement liée à ce message.

### Envoyer un nouveau message
1. Ouvrez l'onglet **Composer**.
2. Sélectionnez le destinataire (l'équipe clinique).
3. Rédigez votre message.
4. Cliquez sur **Envoyer**.

## Astuces et points d'attention
- L'icône cloche en haut de l'application indique le nombre de messages non lus.
