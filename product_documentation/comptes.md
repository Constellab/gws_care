# Comptes clients

## Rôles autorisés
- Opérateur
- Médecin
- Admin
- Responsable de compte (son propre compte uniquement)

## À quoi ça sert
Un compte représente l'entité porteuse d'un ou plusieurs patients : une entreprise (compte Entreprise) ou une personne physique (compte Individuel). Les comptes sont utilisés pour la facturation et le suivi RH.

## Types de comptes
- **Compte Entreprise** : lié à une entreprise cliente, peut regrouper plusieurs patients.
- **Compte Individuel** : lié à une personne physique, pour les patients sans lien entreprise.

## Comment l'utiliser

### Créer un compte
1. Cliquez sur **Comptes** dans le menu de gauche, puis sur **Nouveau compte**.
2. Sélectionnez le type de compte (Entreprise ou Individuel).
3. Renseignez le nom de l'entreprise ou le nom complet, le numéro d'enregistrement, l'adresse et le contact.
4. Astuce : utilisez **Remplir depuis un patient** pour pré-remplir les coordonnées à partir d'un patient existant.
5. Cliquez sur **Créer le compte**.

### Gérer les patients d'un compte
1. Ouvrez la fiche d'un compte.
2. Onglet **Patients** : consultez les patients déjà associés.
3. Cliquez sur **Assigner existant** pour lier un patient déjà créé mais non encore rattaché à un compte.
4. Cliquez sur **Nouveau patient** pour créer directement un patient rattaché à ce compte.

## Astuces et points d'attention
- Un patient doit avoir un compte assigné avant de pouvoir être inscrit à une visite.
