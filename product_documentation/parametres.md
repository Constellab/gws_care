# Paramètres administrateur

## Rôles autorisés
- Admin

## À quoi ça sert
La page Paramètres est réservée aux administrateurs. Elle permet de configurer l'apparence globale de l'application et de gérer les rôles attribués aux utilisateurs Constellab.

## Comment l'utiliser

### Configurer l'apparence (onglet Général)
1. Ouvrez **Paramètres** depuis le menu utilisateur.
2. Onglet **Général** : choisissez la couleur du thème, la langue (Français/Anglais) et téléversez le logo de l'application.
3. Les modifications sont appliquées immédiatement pour tous les utilisateurs connectés.

### Attribuer des rôles à un utilisateur (onglet Rôles utilisateurs)
1. Ouvrez l'onglet **Rôles utilisateurs**.
2. Recherchez l'utilisateur par nom ou email.
3. Cliquez sur un badge de rôle pour le basculer : plein = attribué, contour = non attribué.
4. Pour les rôles **Responsable de compte** et **Patient**, sélectionnez l'entité liée (compte ou patient) après avoir activé le rôle.

### Importer des données en masse (onglet Import)
1. Ouvrez l'onglet **Import**.
2. Suivez la procédure détaillée dans [import-csv.md](import-csv.md).

## Astuces et points d'attention
- Seuls les administrateurs voient cette page ; les autres rôles reçoivent un message d'accès refusé.
