# Résultats d'examens

## Rôles autorisés
- Médecin
- Opérateur

## À quoi ça sert
La section Résultats d'examens permet de saisir les valeurs obtenues pour chaque type d'examen prévu dans une visite (biologie, radiologie, audiométrie, ECG, etc.).

## Comment l'utiliser
1. Depuis le [détail d'une consultation](detail-consultation.md) ou le [détail d'une visite](detail-visite.md), cliquez sur **Ajouter un examen**.
2. Sélectionnez le type d'examen.
3. Saisissez la valeur mesurée.
4. Sélectionnez l'appréciation : Normal, Bas, Haut, Critique bas, ou Critique haut.
5. Joignez un fichier de résultat si disponible (PDF ou image).
6. Enregistrez. Le résultat apparaît immédiatement dans le tableau récapitulatif de la visite.

## Astuces et points d'attention
- L'appréciation est saisie manuellement par l'opérateur ou le médecin et s'affiche avec un code couleur : vert pour Normal, orange pour Bas/Haut, rouge pour les valeurs critiques.
