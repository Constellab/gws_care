# Mon profil

## Rôles autorisés
- Patient

## À quoi ça sert
La page Mon profil affiche vos données personnelles (nom, prénom, date de naissance, sexe, coordonnées) et médicales (poids, taille, numéro de sécurité sociale) enregistrées dans votre dossier.

## Comment l'utiliser
1. Cliquez sur **Mes détails** (ou votre nom) dans le menu utilisateur.
2. Consultez vos informations personnelles et médicales.

## Astuces et points d'attention
- Les informations de votre profil sont gérées par l'équipe médicale. Pour corriger une donnée, contactez la clinique via [mes-notifications.md](mes-notifications.md).
