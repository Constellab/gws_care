# Mode terrain et QR code

## Rôles autorisés
- Médecin
- Opérateur

## À quoi ça sert
Le mode terrain est une interface mobile optimisée pour les opérateurs sur site. Il permet de scanner le QR code d'un patient ou d'un tube pour accéder rapidement à sa fiche et valider sa visite.

## Comment l'utiliser

### Accéder au mode terrain
1. Ouvrez la fiche d'une [campagne](campagnes.md) qui se trouve en phase Terrain.
2. Cliquez sur le bouton **Aller sur le terrain**.

### Scanner un QR code
1. Dans le mode terrain, cliquez sur **Scan caméra** pour activer la caméra de l'appareil.
2. Pointez la caméra vers le QR code de la carte patient ou du tube.
3. La fiche du patient s'ouvre automatiquement après détection.
4. Si la caméra n'est pas disponible, saisissez manuellement le numéro de dossier ou le code du tube dans le champ de recherche.

### Valider une visite sur le terrain
1. Une fois la fiche du patient ouverte, cliquez sur **Marquer la visite sur site comme terminée**.
2. Utilisez le bouton **Absent** si le patient ne s'est pas présenté.
3. Utilisez **Réactiver** pour annuler un statut Absent par erreur.

## Astuces et points d'attention
- Une fois toutes les visites validées ou annulées, utilisez le bouton **Compléter la phase terrain** sur la fiche de campagne pour passer à l'étape d'analyse des résultats.
