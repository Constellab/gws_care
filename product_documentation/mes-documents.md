# Mes documents

## Rôles autorisés
- Patient

## À quoi ça sert
La page Mes documents centralise tous les fichiers médicaux disponibles dans votre dossier : résultats d'analyses, comptes rendus, ordonnances scannées, certificats médicaux.

## Comment l'utiliser
1. Cliquez sur **Mes documents** dans le menu de gauche.
2. Filtrez par type de document (examen, ordonnance, certificat) ou par date.
3. Cliquez sur un document pour l'afficher en aperçu directement dans le navigateur.
4. Cliquez sur **Télécharger le fichier** pour le sauvegarder sur votre appareil.
