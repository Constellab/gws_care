# Visites de campagne

## Rôles autorisés
- Opérateur
- Médecin

## À quoi ça sert
La page Visites de campagne liste toutes les visites liées aux campagnes, tous comptes confondus.

## Comment l'utiliser
1. Cliquez sur **Visites de campagne** dans le menu de gauche.
2. Filtrez par campagne, statut ou date pour retrouver une visite.
3. Cliquez sur une ligne pour ouvrir la fiche de visite et consulter le patient, le statut courant et les actions disponibles.

## Astuces et points d'attention
- Pour le détail des actions possibles sur une visite, voir [detail-visite.md](detail-visite.md).
