# Mes comptes

## Rôles autorisés
- Patient

## À quoi ça sert
Cette page affiche les comptes liés à votre dossier patient : compte entreprise (si votre employeur finance les bilans) ou compte individuel (compte personnel de facturation).

## Comment l'utiliser

### Consulter mes comptes
1. Cliquez sur **Mes comptes** dans le menu de gauche.
2. Consultez le ou les comptes associés à votre dossier, leur type et la date de rattachement.

### Créer un compte personnel
1. Si aucun compte n'est encore associé, cliquez sur **Nouveau compte personnel**.
2. Renseignez vos coordonnées de facturation (nom complet, adresse, contact).
3. Cliquez sur **Créer**.
