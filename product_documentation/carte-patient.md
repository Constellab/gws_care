# Carte patient et QR code

## Rôles autorisés
- Opérateur
- Médecin
- Admin

## À quoi ça sert
La carte patient est un document récapitulatif qui identifie le patient grâce à un QR code unique. Elle est utilisée lors des sessions terrain pour scanner rapidement le dossier.

## Comment l'utiliser

### Générer la carte
1. Ouvrez la [fiche du patient](fiche-patient.md).
2. Cliquez sur l'onglet **Carte patient**.
3. La carte s'affiche avec le nom, le numéro de dossier, la date de naissance et le QR code.

### Imprimer ou télécharger
1. Cliquez sur **Imprimer** pour lancer l'impression directe.
2. Cliquez sur **Télécharger le fichier** pour obtenir un PDF de la carte.

### Utiliser la carte sur le terrain
1. Remettez la carte imprimée au patient ou affichez son QR code à l'écran.
2. L'opérateur terrain scanne ce QR code depuis le mode terrain pour ouvrir instantanément la fiche du patient — voir [terrain-qr.md](terrain-qr.md).
