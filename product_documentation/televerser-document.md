# Téléverser un document

## Rôles autorisés
- Opérateur
- Médecin

## À quoi ça sert
Le téléversement de document permet d'ajouter n'importe quel fichier médical au dossier d'un patient : résultats d'analyses, comptes rendus, radiographies, ordonnances scannées.

## Comment l'utiliser

### Téléverser depuis son ordinateur
1. Depuis la page [Documents](documents.md), cliquez sur **Téléverser un document**.
2. Déposez le fichier (glisser-déposer) ou cliquez pour le sélectionner depuis votre ordinateur.
3. Attendez l'analyse automatique par IA : elle détecte le type de document, la date et le patient associé.
4. Vérifiez et corrigez les informations proposées (patient, type de document, date, description, notes).
5. Cliquez sur **Enregistrer le document**.

### Importer depuis le lab Constellab
1. Sur la même page, choisissez l'option **Importer depuis le dossier lab**.
2. Sélectionnez le dossier (Folder resource) déjà disponible dans le lab Constellab.
3. Tous les documents du dossier sont analysés et proposés automatiquement ; vérifiez puis cliquez sur **Enregistrer tout**.

## Astuces et points d'attention
- Si l'IA ne parvient pas à extraire de texte (document scanné de mauvaise qualité par exemple), renseignez les champs manuellement.
