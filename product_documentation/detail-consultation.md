# Détail d'une consultation

## Rôles autorisés
- Médecin
- Opérateur

## À quoi ça sert
Le détail d'une consultation est l'espace de travail du médecin pour une visite donnée. Il regroupe les résultats des examens, les ordonnances et les certificats d'aptitude.

## Comment l'utiliser

### Enregistrer un résultat d'examen
1. Ouvrez le détail de la consultation depuis la liste des [consultations](consultations.md).
2. Dans la section **Résultats d'examens**, cliquez sur **Ajouter un examen**.
3. Sélectionnez le type d'examen.
4. Saisissez la valeur obtenue.
5. Joignez un fichier de résultat si disponible (PDF ou image).
6. Enregistrez. Le résultat apparaît dans le tableau récapitulatif de la visite.

### Émettre une ordonnance
1. Cliquez sur **Nouvelle ordonnance**.
2. Saisissez le diagnostic ou motif.
3. Ajoutez un ou plusieurs médicaments avec leur posologie, fréquence et durée.
4. Enregistrez. L'ordonnance est immédiatement disponible en téléchargement PDF.

### Émettre un certificat d'aptitude
1. Cliquez sur **Nouveau certificat**.
2. Sélectionnez le type de certificat (aptitude, arrêt de travail, vaccination, etc.).
3. Renseignez la conclusion médicale et les éventuelles restrictions.
4. Enregistrez. Le certificat est archivable et téléchargeable en PDF.
